# CarbonTrack MVP

CarbonTrack is a local, self-contained MVP web app for tracking personal carbon emissions.

## Run

```bash
npm start
```

Then open:

```text
http://localhost:4173
```

## Included

- Onboarding and baseline profile
- Activity logging across transportation, energy, food, shopping, and waste
- Real-time emission calculations with transparent factors
- Dashboard metrics and charts
- Activity filters and undo
- Ranked recommendations and action tracking
- Community challenge mockups
- Profile, privacy controls, export, and reset

Data is stored in browser local storage for this MVP.
